case class TrainingHelper[O, OS, S, SS](input: O, sequenceLengths: ops.Output, timeMajor: Boolean = false, name: String = "RNNDecoderTrainingHelper")(implicit evO: Aux[O, OS], evS: Aux[S, SS]) extends Helper[O, S] with Product with Serializable

RNN decoder helper to be used while training. It only reads inputs and the returned sample indexes are the argmax over the RNN output logits.
